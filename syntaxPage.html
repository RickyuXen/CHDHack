Syntax for Pype

P|[input] -> prints out [input]

Pr|[input] -> prints out [input] in reverse

Pri|[input] -> prints out [input] in reverse in reverse

Prin|[input] -> prints out [input] in reverse in reverse in reverse

Print|[input] -> prints out [input] in reverse in reverse in reverse in reverse

+|[input] -> adds up all numerical values separated by a space, e.g. +|2 3 1 -> 6

-|[input] -> subtracts all numerical values separated by a space, starting from the initial e.g. -|20 5 10 -> 5

*|[input] -> multiplies all numerical values separated by a space, starting from the initial e.g. *|3 2 2 -> 12

/|[input] -> divides all numerical values separated by a space, starting from the initial e.g. /|12 2 3 -> 2

S|[input] -> saves [input]

SS| -> prints out all saved [input]

S(n)| -> prints out nth value saved
